# Atlas Demo Upload Packs

These folders contain synthetic, fictional internal operating exports for the Atlas demo upload flow. They are designed to look like ordinary company finance and operations files while remaining fabricated demo data.

Real companies usually do not keep one perfectly organized scenario file. These packs mimic a CFO data-room bundle assembled from separate system exports: a GL detail export, AP aging report, vendor register, pipeline export, HR planning sheet, security evidence export, and board policy register. The files stay flat inside each folder so a presenter can select the seven files at once, but the file names and columns resemble exports from separate company tools.
